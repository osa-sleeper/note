AWS Cloud9

Cloud9設定

以下の内容で設定を行う。

  1. 「名前」にリソース名を入力する。

  2. 「環境タイプ」で「新しいEC2インスタンス」を設定する。

  3. 「新しいEC2インスタンス」で「t2.micro」を設定する。

  4. 「ネットワーク設定」で「AWS Systems Manager(SSM)」を設定する。

重要

上記の設定で構築した場合、ブラウザのタブを閉じると30分でインスタンスが停止する。
30分以上起動したい場合、「タイムアウト」の値を追加で設定すること。
