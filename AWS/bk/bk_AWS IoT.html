AWS IoT

AWS IoT エンドポイント

  1. 「設定」を選択する。

  2. 「エンドポイント」の値を確認する。

ログレベルの設定

  1. 「ログ」の「ログの管理」ボタンを選択する。

  2. 「ロールを作成」ボタンを選択する。

  3. 「ロール名」にロール名を入力する。

  4. 「ログレベル」で、プルダウンから「Infomational」を選択する。

警告

ログレベルについて深堀する

IoTポリシーの作成

  1. 左のサイドメニューから「セキュリティ」の「ポリシー」を選択する。

  2. 「ポリシーの作成」ボタンを選択する。

  3. 「ポリシー名」に「<名前等>-yyyymmdd-policy」を入力する。

  4. 「ポリシードキュメント」の「ビルダー」が選択されていることを確認する。

  5. 「ポリシー効果」で「許可」を選択する。

  6. 「ポリシーアクション」にすべての「AWS IoT アクション」の「*」を選択する。

  7. 「ポリシーリソース」に「*」を入力する。

  8. 「作成」ボタンを選択する。

警告

ポリシードキュメントについて深堀する

モノの作成

  1. 左サイドメニューから「すべてのデバイス」の「モノ」を選択する。

  2. 「モノを作成する」を選択する。

  1. 「１つのモノを作成」を選択する。

  2. 「次へ」を選択する。

5. 「モノの名前」を設定する。（後で参照するのでメモしておく。） 6.「名前のないシャドウ（クラシック）」を選択する。 7.「次へ」を選択する。

警告

Device Shadowの選択肢の違いを確認する

  1. 「新しい証明書を自動生成（推奨）」を選択する

  2. 「次へ」を選択する

  1. 先行手順で作成済みのポリシーにチェックを入れる

  2. 「モノを作成」を選択する。

  1. 以下のファイルをダウンロードする。
