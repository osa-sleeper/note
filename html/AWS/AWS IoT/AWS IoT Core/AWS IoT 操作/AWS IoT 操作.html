AWS IoT Core 操作

デバイスシャドウ

クラウド上でデバイスシャドウに対して設定値を指定することにより、デバイスが設定値を同期して、デバイスの遠隔設定ができる。

デバイスシャドウの更新（メッセージ送信頻度の遠隔設定）

  1. 「すべてのデバイス」の「モノ」を選択する。

  2. 「作成したモノ（xxxx-mono-property-yyyymmdd）」を選択する。

  3. 画面下部のタブから「Device Shadow」を選択する。

  4. 表示されたリストから「Classic Shadow」を選択する。

  1. 「Device Shadow ドキュメント」の「編集」ボタンを選択する。

  2. 表示された編集画面で、JSON内容を以下のように変更する。

{
    "state": {
        "reported": {
            "wait_time": 5
        },
        "desired": {
            "wait_time": 1
        }
    }
}

注釈

「desired」内に「"welcome": "aws-iot"」というプロパティ含まれているときは、「"welcome": null」で削除可能である。

{
    "state": {
        "reported": {
            "wait_time": 5,
            "welcome": "aws-iot"

        },
        "desired": {
            "wait_time": 1,
            "welcome": null
        }
    }
}

注釈

シャドウドキュメントのうち 「desired」がデバイスに指示している状態、 「reported」がデバイスから報告された状態である。 「desired」と「reported」の状態に差異がある場合、「delta」というフィールドに差分が表示される。
